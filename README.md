#Sistema de Control de Sala de Cine

Programa en Python que permite registrar personas que ingresan a una sala de cine

## Funcionamiento
1. Solicita la capacidad total de la sala.
2. Registra personas que ingresan.
3. Pide la edad de cada Persona.
4. Clasifica en:
- Niño ( 0 - 12)
- Adulto (13-59)
- Adulto Mayor (60 o màs)

## Resultados finales

El sistema muestra:

- Total de Personas Ingresadas
- Cantidad de Niños
- Cantidad de Adultos
- Cantidad de Adultos Mayores
- Indica si la sala se llenò o no

## Conceptos aplicados
- Ciclos con lìmite (while)
- Contadores
- Condicionales
